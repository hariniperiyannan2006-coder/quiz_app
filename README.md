        Quiz App (Python)
##Description
This project is a simple quiz application developed using Python.
It asks multiple questions to the user, checks the answers,
and calculates the final score.

##Features
- Displays quiz questions
- Accepts user answers
- Calculates and displays final score
- Simple console-based interaction

##Technologies Used
- Python 3

##How to Run
1. Download quiz_app.py
2. Open terminal / command prompt
3. Run:
   python quiz_app.py

##Learning Outcome
- Conditional statements
- Loops
- User input handling
- Basic control flow
